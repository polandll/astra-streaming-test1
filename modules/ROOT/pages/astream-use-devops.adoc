// tag::set-bearer-token[]
You need to set the bearer token with the token you downloaded in the `client.conf` file before running any commands:
[source,shell,subs="attributes+"]
----
export BEARER_TOKEN={bearer_token_value}
----
// end::set-bearer-token[]

// tag::get-pulsar-clusters[]
[#get-pulsar-clusters]
== Get Pulsar clusters

To get all the Pulsar clusters that currently exist, execute the command:
[tabs]
====
cURL command::
+
--
[source,shell,subs="attributes+"]
----
include::astra-streaming-code-ex:ROOT:example$get_clusters.sh[]
----
--

Result::
+
--
[source,plaintext]
----
include::example$result/get_clusters.result[]
----
--
====
// end::get-pulsar-clusters[]
